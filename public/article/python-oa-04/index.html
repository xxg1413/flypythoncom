pdfplumber介绍

Pdfplumber是一个可以处理pdf格式信息的库。它可以查找关于每个文本字符、矩阵、和行的详细信息，也可以对表格进行提取并进行可视化调试。

官方repo:
https://github.com/jsvine/pdfplumber

安装：
pip install pdfplumber

常用方法
